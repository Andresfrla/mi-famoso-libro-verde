# Politicas Generales
Estas politicas describen las practicas basicas para el uso publico de este servicio. Se recomienda adaptar estas secciones a la legislacion aplicable en tu region.

## Politica de Datos
Esta politica describe como recogemos, usamos y protegemos los datos personales de los usuarios.

Datos que recogemos
- Identificadores proporcionados por el usuario (nombre, correo)
- Datos de uso (pagina visitada, tiempo en sitio, acciones)
- Datos tecnicos (direccion IP, tipo de dispositivo, navegador)
- Cookies y tecnologias similares

Uso de los datos
- Proveer, mantener y mejorar el servicio
- Personalizar la experiencia del usuario
- Comunicaciones operativas (soporte, seguridad, notificaciones)

Retencion y comparticion
- Conservamos datos solo por el tiempo necesario
- Podemos compartir datos con terceros para operacion del servicio o por requerimientos legales
- Compartir datos fuera de la region sujeta a acuerdos y protecciones aplicables

Seguridad
- Implementamos controles tecnicos y procedimientos razonables
- Limitamos el acceso a datos a personal autorizado

Derechos de usuarios
- Acceder, corregir o eliminar tus datos personales
- Retirar consentimiento cuando aplica
- Presentar reclamaciones ante autoridades competentes

Cambios a la politica
- Podemos actualizar esta politica; revisa la fecha de ultima modificacion
- Notificaremos a los usuarios sobre cambios significativos
